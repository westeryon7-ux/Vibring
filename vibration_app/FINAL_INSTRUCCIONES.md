# App de Vibración - Proyecto Completo

## Descripción
Esta aplicación permite hacer vibrar otro dispositivo cuando se toca un botón en tu dispositivo, y viceversa. Perfecta para parejas que quieren conectarse de forma divertida y sencilla.

## Características
- Interfaz intuitiva con pantalla de configuración inicial
- Introducción de datos de Firebase a través de la interfaz (no requiere editar código)
- Emparejamiento bidireccional de dispositivos
- Vibración inmediata entre dispositivos conectados
- Diseño responsive y amigable

## Componentes del Proyecto

### Archivos Principales:
1. **App.js** - Código principal de la aplicación con interfaz de configuración
2. **firebaseConfig.js** - Plantilla de configuración (no requiere edición)
3. **package.json** - Dependencias del proyecto
4. **app.json** - Configuración de la aplicación Expo

### Documentación:
1. **PASO_A_PASO.md** - Instrucciones detalladas clic por clic
2. **GENERAR_APK.md** - Guía para crear el archivo APK
3. **README.md** - Documentación general del proyecto

## Cómo Usar

### Para Usuarios Finales (sin conocimientos de programación):
1. Sigue las instrucciones en PASO_A_PASO.md
2. Crea un proyecto en Firebase (solo requiere clics, no programación)
3. Genera el APK o instala la app en ambos dispositivos
4. Introduce los datos de Firebase en la interfaz de la app
5. Comparte los IDs de dispositivo entre ambos usuarios
6. ¡Listo! Ya pueden hacer vibrar los dispositivos mutuamente

### Funcionalidad:
- Pantalla de configuración inicial donde se introducen los datos de Firebase
- Cada dispositivo genera un ID único para la identificación
- Campo para introducir el ID del dispositivo emparejado
- Botón para hacer vibrar el otro dispositivo
- Botón para probar vibración local
- Opción para cambiar la configuración en cualquier momento

## Requisitos
- Dispositivos Android o iOS
- Conexión a Internet en ambos dispositivos
- Cuenta de Google para crear proyecto en Firebase

## Seguridad
- Los datos de Firebase se introducen localmente en cada dispositivo
- No se almacenan credenciales en servidores externos
- Cada emparejamiento es único y requiere intercambio manual de IDs

## Soporte
Si tienes problemas:
1. Revisa las instrucciones en PASO_A_PASO.md
2. Asegúrate que ambos dispositivos estén conectados a Internet
3. Verifica que los IDs estén correctamente introducidos
4. Confirma que los datos de Firebase sean correctos en ambos dispositivos
