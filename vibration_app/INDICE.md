## Descripción de la Aplicación

La aplicación permite hacer vibrar otro dispositivo cuando se toca un botón en tu dispositivo, y viceversa. Tiene una pantalla de configuración inicial donde se introducen los datos de Firebase y los IDs de los dispositivos emparejados.

### Características:
- Interfaz intuitiva con pantalla de configuración inicial
- Introducción de datos de Firebase a través de la interfaz (no requiere editar código)
- Emparejamiento bidireccional de dispositivos
- Vibración inmediata entre dispositivos conectados
- Diseño responsive y amigable

### Funcionalidad:
1. Al abrir la app por primera vez, se muestra una pantalla de configuración
2. Se introducen los datos de Firebase y el ID del dispositivo emparejado
3. Una vez configurado, se puede hacer vibrar el otro dispositivo con un botón
4. La app también vibra cuando el otro dispositivo envía la señal

## Notas Importantes

- La aplicación está lista para usar, solo requiere configuración inicial
- No se requiere experiencia en programación para usarla
- Ambos dispositivos deben tener conexión a Internet
- Se requiere una cuenta de Google para crear el proyecto en Firebase
